# Docstring Generation Prompt Template

**Role**: Technical Writer
**Task**: Generate a complete and accurate docstring for the given [LANGUAGE] function.
**Input Placeholder**: [FUNCTION_CODE]
**Expected Output**: A properly formatted docstring describing the function purpose, parameters, return value, and any exceptions raised.

---

## Example Prompt

You are a technical writer specializing in code documentation.
Generate a complete docstring for the following [LANGUAGE] function.
Use [DOCSTRING_FORMAT] format (e.g., Google, NumPy, JSDoc).

```
[FUNCTION_CODE]
```

Include: function description, all parameters with types, return value with type, and any exceptions that may be raised.
