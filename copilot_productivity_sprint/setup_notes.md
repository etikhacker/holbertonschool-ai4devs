# Setup Notes – AI Coding Assistant Environment

## IDE
- **Editor**: Visual Studio Code
- **Version**: 1.99.3
- **Platform**: Windows 11

## Extensions Installed
- **GitHub Copilot** v1.256.0 — AI-powered code completion and suggestions
- **GitHub Copilot Chat** v0.26.0 — inline chat for code explanations and generation
- **Prettier** v11.0.0 — code formatter for consistent style
- **ESLint** v3.0.10 — JavaScript/TypeScript linting
- **Python** v2024.6.0 — Python language support
- **GitLens** v16.3.0 — enhanced Git integration

## GitHub Copilot Setup Steps
1. Opened VS Code and navigated to Extensions (Ctrl+Shift+X)
2. Searched for "GitHub Copilot" and clicked Install
3. Installed "GitHub Copilot Chat" extension as well
4. Clicked "Sign in to GitHub" when prompted
5. Authorized VS Code to access GitHub account (etikhacker)
6. Verified Copilot icon appeared in the status bar at the bottom
7. Opened a new Python file and confirmed inline suggestions appeared while typing

## Notes
- GitHub Copilot requires an active GitHub account with Copilot access
- Free tier available for verified students via GitHub Education
- Copilot suggestions can be accepted with Tab and dismissed with Escape
- Copilot Chat can be opened with Ctrl+Shift+I for the sidebar panel
