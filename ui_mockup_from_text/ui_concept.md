# UI Concept - EduTrack

## Purpose
Help students organize assignments, track grades, and follow AI-generated study plans through a clean and intuitive academic management interface.

## Users
- Students: manage assignments, view grades, and follow personalized study plans
- Teachers: create assignments, monitor student progress, and track attendance

## Core Screens
- Dashboard with upcoming assignments and grade summary
- Assignment list with due dates and priority labels
- Grade tracker with GPA calculator and performance chart
- AI Study Plan screen with weekly schedule
- Attendance management screen for teachers
- Settings screen with language and theme options

## Actions
- Create and submit assignments
- Enter and calculate grades
- Generate and view AI study plan
- Mark attendance as present, absent, or late
- Toggle dark mode and change language
- View and export progress reports
