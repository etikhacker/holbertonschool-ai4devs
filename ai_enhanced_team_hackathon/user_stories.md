# User Stories
## AI Study Plan Generator

## User Story 1
As a student, I want to register an account so that I can save my study plans and track my progress securely.

## User Story 2
As a student, I want to input my subjects and their exam dates so that the AI can understand my upcoming deadlines.

## User Story 3
As a student, I want to specify how many hours per day I can study so that the AI generates a realistic and achievable schedule.

## User Story 4
As a student, I want the AI to generate a personalized day-by-day study plan so that I know exactly what to study each day.

## User Story 5
As a student, I want to mark individual study topics as completed so that I can track my progress through the plan.

## User Story 6
As a student, I want to receive AI-generated study tips for each subject so that I can study more effectively and efficiently.

## User Story 7
As a student, I want to regenerate my study plan with updated inputs so that I can adjust when my schedule or priorities change.

## User Story 8
As a student, I want to view a progress dashboard showing completed and remaining topics so that I can see how prepared I am for each exam.

## User Story 9
As a student, I want to delete a study plan I no longer need so that my dashboard stays organized and relevant.

## User Story 10
As a student, I want to log in and access my saved study plans from any device so that I can study anywhere without losing my progress.
