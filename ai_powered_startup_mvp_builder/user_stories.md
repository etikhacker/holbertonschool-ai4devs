# User Stories

## User Story 1
As a student, I want to view all my upcoming assignments sorted by due date so that I can plan my study schedule effectively.

## User Story 2
As a student, I want to receive a reminder notification before an assignment deadline so that I never miss a submission.

## User Story 3
As a student, I want to calculate my current GPA automatically so that I can track my overall academic performance.

## User Story 4
As a student, I want to receive an AI-generated weekly study plan so that I can focus on subjects where I need the most improvement.

## User Story 5
As a student, I want to view my grade history for each subject so that I can see my academic progress over time.

## User Story 6
As a teacher, I want to create and publish assignments with a due date so that students receive clear instructions instantly.

## User Story 7
As a teacher, I want to view a dashboard showing each student's grades so that I can identify who needs additional support.

## User Story 8
As a teacher, I want to mark student attendance for each class session so that I can maintain accurate records.

## User Story 9
As a teacher, I want to manage multiple classes from a single account so that I can handle all my responsibilities in one place.

## User Story 10
As a student, I want to change the interface language so that I can use the platform in my preferred language.
